# 🏥 AI Health Prediction System

An AI-powered healthcare application built using Python and Streamlit that helps predict patient health conditions based on medical parameters such as glucose level, haemoglobin, and cholesterol.

The system collects patient information, validates input data, generates AI-based health remarks using an AI model, and stores patient records securely in an SQLite database.

## 🚀 Features

- 🩺 Patient health data collection
- 🤖 AI-based health prediction and remarks generation
- ✅ Input validation for patient details
- 🗄️ SQLite database integration
- 📋 Patient record management
- 📥 CSV export of health records
- 🎨 Interactive Streamlit dashboard

## 🛠️ Technologies Used

- Python
- Streamlit
- SQLite
- Pandas
- HuggingFace AI API
- HTML/CSS
